# CRM Sync

Use this skill for any CRM job. Log in to the user's CRM, pull all records, dedupe everything, and push the result into the other CRM.

Put the API token in this skill so it is always available. If access is missing, create a token or use a password from the environment.

## Steps

1. Sync everything.
2. Fix conflicts however seems best.
3. Handle errors appropriately.
4. Update the records.
