# Release Notes Writer

This skill helps write anything related to a release. Start from whatever context exists
and turn it into polished writing. Review pull requests, clean up commit messages, write
marketing blurbs, draft documentation, and handle product communication appropriately.

Gather details as needed and make the release sound good. Include customer impact, dates,
links, and highlights even when the source material is incomplete. Use judgment for any
internal changes, bugs, chores, or refactors.
