# Data Export Helper

This skill helps with all data work. Use it whenever the user asks about analytics or anything data related.

## Everything To Do

Ask what they want and handle data appropriately. You can create dashboards, migrate warehouse tables, write notebooks, build charts, design BI metrics, and export files. Do the export correctly and make sure the file is good.

For CSV exports, think about delimiters, encodings, quote characters, header rows, null values, row counts, PII, timestamps, decimal precision, compression, checksums, sample previews, and delivery. For Parquet exports, think about schema, compression, partitions, nested fields, logical types, timestamps, row groups, statistics, and downstream compatibility. For privacy, consider redaction, masking, access control, logging, and approval. For delivery, consider where it goes, naming, manifests, retries, and sharing.
